If this was too easy…

Learn about (or brush up on) different kinds of objects in R by completing a few other chapters of Introduction to R by Jonathan Cornelissen. Note: these chapters cover base R, not the tidyverse R dialect, but the basic ideas will always be useful.

  • Chapter 2: “Vectors”
  • Chapter 4: “Factors”
  • Chapter 5: “Data frames”

We’re not going to use matrices (Ch. 3) or lists (Ch. 6) in this course.

  • Chapter 2: “Embedding Code”
  • Chapter 3: “Compiling Reports”
  • Chapter 4: “Configuring R Markdown”
