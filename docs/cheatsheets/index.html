Cheat Sheets

Cheat sheets will provide a brief introduction to a range of data science tools, many of which we will continue to use regularly throughout the course modules. Tutorial sessions will be woven in between modules and are meant to fit within a single class session. While modules aim to help you master fundamental skills through a hands-on analysis over several weeks, tutorials are meant primarily to give you some context and a flavor for what is possible.
