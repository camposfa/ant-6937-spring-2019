Setup

To complete this week’s activities, you must have a few packages installed. You can do so by running the commands below in your R console (only needs to be done once).

install.packages("tidyverse")
install.packages("here")
install.packages("rmarkdown")

Complete in Class

Markdown CV

Instructions:

  1. Create a basic CV in markdown (you can make stuff up!)

  2. What to include

    1. Top-level header should be your name

    2. Include contact info: phone, email, and link to some website.

    3. Include second-level headings for education, employment, publications, and teaching

    4. Each section should include a bulleted or numbered list of items, or a table

    5. Highlight years in bold and (possibly fake) journal names in italics

    6. Embed some kind of image at the bottom

  3. Submit the .md and .html files for your completed CV in Blackboard.
